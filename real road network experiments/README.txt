Download the datasets of 12 cities from the below link:
https://figshare.com/articles/Urban_Road_Network_Data/2061897

Unzip them and put the "x_edgelist.txt" files in the root directory. 

By running the code, the Robust Minimum Cost Flow Problem Under Multiple Ripple Disruptions will be solved over datasets one by one. No action needed!

You can easily change the number of disruptions and other parameters just by changing the value of associated varibales in the code.

The results are automatically recorded in a ".csv" file in the root directory.
